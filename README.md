# vitela

vitela is a small browser-based image tool that turns layered images into a soft paper-like composition.

## What it does

- Load one or more images
- Preview them on a canvas
- Adjust blur, grain, ink, and paper settings
- Reorder layers
- Save and load presets
- Export the final result as a PNG

## How to use

1. Open `index.html` in a browser.
2. Load your images.
3. Adjust the settings until the look feels right.
4. Export the image when you are done.

## Files

- `index.html` - app structure
- `styles.css` - visual style
- `script.js` - canvas rendering and controls

## Notes

The project runs entirely in the browser and does not need a build step.
